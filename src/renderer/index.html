AI USAGE
Settings
Claude session

Paste your claude.ai sessionKey cookie. Get it from DevTools → Application → Cookies → claude.ai → sessionKey.

Open claude.ai → Settings → Usage ↗ Clear claude.ai session
Gemini session

Paste your gemini.google.com cookies. Easiest: DevTools → Network → click any request → Request Headers → copy the whole Cookie: value and paste it here (Google needs several cookies, not just one).

Open gemini.google.com ↗ Clear Google session
ChatGPT session

Paste your chatgpt.com __Secure-next-auth.session-token cookie (DevTools → Application → Cookies → chatgpt.com), or the whole Cookie: header. Note: ChatGPT does not publish usage %, so only your plan + connection are shown.

Open chatgpt.com ↗ Clear ChatGPT session
